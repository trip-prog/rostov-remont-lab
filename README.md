# ROSTOV REMONT

Лендинг компании по ремонту квартир под ключ в Ростове-на-Дону. Светлая премиальная тема (белый + горчичное золото).

**Демо:** https://trip-prog.github.io/rostov-remont/

## Секции
- Шапка со sticky-навигацией и мобильным меню
- Hero с оффером, гарантийным бейджем и блоком преимуществ
- О компании + статистика
- Услуги (6 видов ремонта)
- Портфолио выполненных проектов
- Пять этапов работы
- Отзывы клиентов
- Форма заявки «Заказать звонок» с маской телефона (демо-режим)
- Футер с контактами

## Стек
Чистые HTML + CSS + JS, без сборки и зависимостей. Шрифты Manrope и Playfair Display, адаптив, scroll-reveal, `prefers-reduced-motion`.

## Запуск
Открыть `index.html` в браузере — сборка не требуется.

> Демонстрационный проект. Форма заявки не отправляет данные на сервер.
